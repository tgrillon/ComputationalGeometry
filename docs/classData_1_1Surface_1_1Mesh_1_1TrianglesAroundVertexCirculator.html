Detailed Description

Circulator to iterate over the triangles around a given vertex.

Note
The circulator will iterate in counter-clockwise direction first, then in clock-wise direction if it reaches a boundary.
Case where the one-ring neighborhood is closed.
Case where the one-ring neighborhood is opened.
